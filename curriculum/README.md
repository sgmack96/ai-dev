# Cloudflare SME Curriculum
## Complete Dashboard Mastery — Theory + Implementation + Demo Readiness

> **Goal:** Become a Subject Matter Expert on every Cloudflare product visible on the dashboard.  
> **Lab Domain:** `macksportreport.com`  
> **Pacing:** 1-2 modules/day (~5 weeks total)  
> **Format:** Each module = Theory (SE-level) + Deep Dive (Architect-level) + Lab + Demo Script

---

## How to Use This Curriculum

1. Work through modules in order — each phase builds on the previous
2. Complete the **Hands-On Lab** before moving to the next module
3. Record your **Self-Check** answers at the bottom of each workbook
4. After each phase, run the **Demo Script** cold — no notes
5. Mark modules complete with `[x]` as you go
